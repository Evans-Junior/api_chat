# Message Application

This is a simple web application for sending and receiving messages between producers and farmers.

## Features

- Send messages from producer to farmer and vice versa.
- View sent messages.
- Messages are displayed with timestamps.
- Responsive design.

## Technologies Used

- HTML
- CSS
- JavaScript (jQuery)
- PHP
- MySQL
